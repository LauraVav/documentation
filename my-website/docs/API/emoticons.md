# Emoticons

List all emoticons found.

| KEY        	| TYPE           	| DESCRIPTION                          	|
|------------	|----------------	|--------------------------------------	|
| confidence 	| float          	| Confidence in value matching         	|
| emoticon   	| EmoticonObject 	| boolean values for matched emoticons 	|
